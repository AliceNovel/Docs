package.jsonの作成方法

package.jsonの作成方法に関する資料です。

はじめに

package.jsonはゲームの初期設定を決定する重要な役割を果たしています。

package.jsonはanprojファイル正式採用となるv0.9.0-rc1以降で利用可能です。

game-name

ゲームタイトルを設定可能です。

記述: 推奨

初期設定: "" (タイトルなし)

例

{

  "game-name: "Alice wonder land"

}

利用可能バージョン

Alice Novel: v0.9.0-rc1 ~

first-read

初回ロード時に読み込むファイルを設定できます。

記述: 推奨

初期設定: "main.anov"

例

{

  "first-read": "story/main.anov"

}

利用可能バージョン

Alice Novel: v0.9.0-rc1 ~

root

種類別のファイルのルートを設定できます。

記述: 任意

初期設定(root-image): "image/"

初期設定(root-background): "image/background/"

初期設定(root-story): "story/"

初期設定(root-data): "data/"

初期設定(root-chara): "character.json"

初期設定(root-save): "save/"

初期設定(root-audio): "audio/"

初期設定(root-movie): "movie/"

例

{

  "root-image": "img/"

  "root-background": "img/bg/"


  "root-story": "story/"

}

利用可能バージョン

Alice Novel: ? ~
